Our Projects

At our coding club, we believe in learning by doing. Here are some of the exciting projects we've worked on:

🎮 Snake Game

Test your reflexes with our classic Snake Game! Built using JavaScript and HTML5 Canvas, this game is a fun way to learn about game logic and animation.

🌐 Personal Portfolio Websites

Our members have designed and developed personal portfolio websites to showcase their skills, projects, and achievements. These sites are built with HTML, CSS, and JavaScript, featuring interactive elements and responsive design.

🤖 AI Chatbot

We've developed an AI-powered chatbot that can assist users with coding-related questions. Using Python and Natural Language Processing (NLP), the chatbot provides real-time responses and learns from user interactions.

📊 Data Visualization Dashboard

Our team built a dynamic dashboard that visualizes data trends using Python, R, and Power BI. This project helps users analyze and interpret large datasets efficiently.

📱 Mobile To-Do List App

We created a mobile-friendly To-Do List app using React Native. It allows users to manage tasks efficiently with reminders and progress tracking.

We are constantly working on new and exciting projects. Join us and be part of the innovation!
